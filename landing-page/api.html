TriadFramework

The main framework class for coordinating multi-agent AI systems with structured oversight.

class TriadFramework()

Core framework implementing multi-agent coordination principles. This class ensures all AI agents operate within proper system constraints and structured accountability mechanisms.

__init__()

Constructor
def __init__(self) -> None

Initialize a new TriadFramework instance with default system principles and empty coordination session.

Example
from triad import TriadFramework

# Initialize the framework
framework = TriadFramework()
print(f"Framework initialized with {len(framework.system_principles)} principles")

process_request

async method
async def process_request(self, request: str, deps: TriadDeps, require_consensus: bool = False) -> Dict[str, Any]

Process a request through the four-agent framework with proper oversight and validation.

Parameters
request
str
The user request to process through the framework
deps
TriadDeps
Framework dependencies and configuration
require_consensus
bool
Whether all agents must agree on the result (default: False)
Returns
Dict[str, Any]
Result dictionary containing:
  • result: The processed response
  • agents_involved: List of agents that participated
  • system_compliance: Boolean indicating system compliance
  • audit_trail: Dictionary of agent actions and decisions
Example
import asyncio
from triad import TriadFramework, TriadDeps

async def main():
    framework = TriadFramework()
    deps = TriadDeps.from_environment()
    
    result = await framework.process_request(
        request="Analyze customer feedback trends",
        deps=deps,
        require_consensus=True
    )
    
    print("Response:", result["result"])
    print("Compliance:", result["system_compliance"])

asyncio.run(main())

AgentRole

Enumeration of available agent roles in the framework.

class AgentRole(str, Enum)

Defines the four specialized roles available in the AI Triad Model framework.

Values

PLANNER
"planner"
Strategic planning and task coordination
EXECUTOR
"executor"
Task execution and operational management
EVALUATOR
"evaluator"
Quality review and compliance checking
OVERWATCH
"overwatch"
System monitoring and crisis management
Example
from triad import AgentRole

# Use in agent configuration
agent_role = AgentRole.PLANNER
print(f"Agent role: {agent_role.value}")  # Output: planner

# Check role type
if agent_role == AgentRole.PLANNER:
    print("This is a planning agent")
